HTML Basics 🐣

HTML和CSS是最基本的网页构成元素。HTML负责记述网页结构和内容，CSS负责记述这些内容的排版，字体，及风格等。

HTML原子的基本结构

HTML的构成原子，是一个一个的tag语句。这些语句的基本构成元素是：
  1. 起始tag
  2. 结束tag（有些tag是不需要结束的，比如<img>，<br>，这种叫做self-closing）
  3. 内容
  4. 属性（optional）

    <!-- 以下是一个简单的p元素 -->
    <!-- class是这个tag的一个属性;实用中class常被用来分组各种元素 -->
    <p class="article"> Be real. </p>

    <!-- 比如在CSS中可以这样抓取同为article组的元素 -->
    .article{
      font-size : 16px;
    }
所谓的属性(attribute)也就是上面例子中的class,或者经常会用到的id, style = "..."之类，主要是用于给tag做区分，便于CSS和Javascript抓取。 注意属性在起始tag之内，与tag名用空格隔开，属性的值最好用引号包括起来。

HTML网页的Boilerplate

Boilerplate就是一个基础模板，在很多现代编辑器中都有很便捷的快捷键对应。比如在Visual studio code里，只要在一个空白的html文件中键入!并键盘输入Tab，就会返回如下的模板： 

  <!DOCTYPE html>
  <html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
  </head>
  <body>
    
  </body>
  </html>

1.开头的DOCTYPE是告诉浏览器html版本的声明，在过去有很多版本林立的时候，需要写HTML VER 4.0之类复杂的声明，但是HTML5以后这一切都被简化， 现在只需要写html即可，默认是HTML5.

2.<head>中存放的是一些关于网页的管理信息，比如网页的标题，CSS文件和Javascript文件的位置等

3.<meta>中的charset是一个重要的设定，决定了文件的编码格式。现在绝绝大多数网页都采用了UTF-8编码，可以对应基本所有的 人类语言。在一些比较旧的日语或者中文网页偶尔会发现浏览器乱码，原因就是网页采用了比较旧的编码格式，那些编码对应到UTF-8就是一些 奇怪的符号，所以就产生了文字化け。另外，UTF-8还包括了很多emoji，比如本页的标题:)

4.其他还有一些meta标签，比如description，keyword之类，也是帮助浏览器认识网页用的，Google会利用这些标签来帮助其搜索内容。

CSS的原子 CSS的最基本组成元素就是一个一个的rule set。每个rule都有一个selector来指定其作用的对象，至少一个属性名称和一个属性值。属性名称和属性值用:隔开。你也可以同时在一个rule set中指定复数个对象，比如 ```css p, li, h1 { color: red; width: 500px; border: 1px solid black; } ``` 注意复数个对象之间，用逗号隔开。 #### Selector的种类 Selector并不一定是HTML中的tag名，也可以是属性或者其他值，而且这些条件都可以进行叠加，以方便对特定的元素进行定点的定制。比如 ```css table td{ color : red; } ``` 这样的句式就是对所有table中的td元素进行指定(注意这里table和td之间是空格，不是逗号)。当然CSS还有很多很多可以指定的Selector，具体可以参考[MDN开发者档案](https://developer.mozilla.org/en-US/docs/Learn/CSS/Introduction_to_CSS/Selectors)。 ### 2. 字体 为了使用一种新的字体，我们首先需要在html文件中链接这些字体。比如说，添加如下链接到index.html的head段就可以利用Google的免费字体OpenSans。 ```html ``` 然后，我们就可以在css中指定使用这种字体，比如： ```css html { font-size: 10px;
